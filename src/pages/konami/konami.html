Código de Konami

En 1986 Gradius salió para la NES, en este juego, Kazuhiso Hashimoto quiso implementar un método de Debug para poder desarrollar de forma eficiente.
una vez activado, otorgaba al jugador una gran cantidad de mejoras con las que era mucho más fácil progresar en los nivéles.
El código se hizo muy famoso, ya que a la hora de publicar el juego, por un descuido, se les olvidó desactivar esta opción y todo el mundo podía usarlo.
